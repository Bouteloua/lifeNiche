Experiments

Lifemapper RAD experiments are containers for the data you will use for macroecological experiments.

Count Experiments

The "countExperiments" function allows you to count the number of Lifemapper RAD experiments that match the parameters you specify. Information about these parameters can be found in the code documentation.

Example: Count RAD experiments with EPSG code 4326 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cl.rad.countExperiments(epsgCode=4326) 3 >>> cl.close()

Get Experiment

The "getExperiment" function allows you to retrieve an experiment

Example: Get experiment 12345 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> exp = cl.rad.getExperiment(12345) >>> cl.close()

List Experiments

The "listExperiments" function returns a list of RAD experiments that match the specified parameters. More information about these parameters can be found in the code documentation.

Example: List 20 RAD experiments >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> exps = cl.rad.listExperiments(perPage=20) >>> cl.close()

Post Experiment

The "postExperiment" function creates a new RAD experiment. It takes a name parameter and an EPSG code to start.

Example: Post a new experiment named 'MyExp' with EPSG 3410 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> exp = cl.rad.postExperiment('MyExp', 3410) >>> cl.close()

Buckets

Lifemapper Range and Diversity buckets are containers for calculations based on a certain extent and resolution. Experiments can have many buckets to explore the effects of scale.

Count Buckets

The "countBuckets" function counts the number of buckets that match the specified criteria. Information about the available parameters can be found in the code documentation.

Example: Count all buckets for experiment 123 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> count = cl.rad.countBuckets(123) >>> cl.close()

Get Bucket

The "getBucket" function returns metadata for the chosen bucket. It takes the id of the bucket and the id of its experiment for input.

Example: Get bucket 123 for experiment 456 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> bkt = cl.rad.getBucket(456, 123) >>> cl.close()

Get Bucket Shape Grid Data

The "getBucketShapegridData" function allows you to retrieve the shape grid attached to a bucket. You can optionally retrieve the shape grid with values from the intersected presence / absence layers.

Example: Get bucket 123 of experiment 456's shape grid intersected. Store it at /tmp/shapeGridData.zip >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cl.rad.getBucketShapegridData('/tmp/shapeGridData.zip', 456, 123, intersected=True) >>> cl.close()

List Buckets

The "listBuckets" function returns a list of buckets that match the parameters specified. Information about these parameters can be found in the code documentation.

Example: List the buckets of experiment 123 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> bkts = cl.rad.listBuckets(123) >>> cl.close()

Add bucket

The "addBucket" function adds a new bucket to an experiment. It will build a new shape grid for this bucket based on the parameters you specify.

Example: Add a bucket to experiment 123 with the name 'MyShape' using hexagonal cells of size 0.5 decimal degrees for EPSG 4326 and for the western hemisphere >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> bkt = cl.rad.addBucket(123, 'MyShape', 'hexagon', 0.5, 'dd', 4326, '-180,-90, 0, 90') >>> cl.close()

Intersect Bucket

The "intersectBucket" function performs an intersection of the cells in the bucket's shape grid with the presence / absence layers of the experiment.

Example: Request an intersection for bucket 123 of experiment 456 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cl.rad.intersectBucket(456, bucketId=123) >>> cl.close()

Randomize Bucket

The "randomizeBucket" function sends a request to randomize a bucket which will result in a new randomized pam sum. There are currently two methods for randomizing a bucket, swap and splotch. Additionally, the swap randomization algorithm takes a parameter for the number of iterations to perform.

Example: Randomize bucket 123 of experiment 456 using splotch >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cl.rad.randomizeBucket(456, 123, method='splotch') >>> cl.close()
Example: Randomize bucket 123 of experiment 456 using swap and 10000 iterations >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cl.rad.randomizeBucket(456, 123, method='swap', iterations=10000) >>> cl.close()

Presence Absence Layers

Presence / Absence layers are used to create PAMs. They are a pairing of an uploaded layer and a set of presence / absence values indicating what should be considered presence or absence.

Add Presence / Absence Layer

The "addPALayer" adds a new presence absence layer to an experiment. The same layer can have multiple presence / absence values associated with it. Note: Use 'pixel' for the presence attribute of a raster layer

Example: Add presence / absence values for the raster layer 123 to the experiment 456. Set the minimum presence to be 100 and the max to be 254 and 40% coverage to be considered present. >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cl.rad.addPALayer(456, 123, 'pixel', 100, 254, 40) >>> cl.close()

Get Presence / Absence Layers

The "getPALayers" function returns the presence absence layers associated with an experiment

Example: Get the PA layers for experiment 1234 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> paLyrs = cl.rad.getPALayers(1234) >>> cl.close()

PAM Sums

PAM Sums (Presence Absence Matrices and Summaries) are used to create the statistics and analysis for the Lifemapper Range and Diversity module. They are matrices of sites on one axis and presence or absence for a layer on the other axis.

Count PAM Sums

The "countPamSums" function counts the number of PAM Sums that match the specified criteria. Information about the available parameters can be found in the code documentation.

Example: Count all pam sums for bucket 123 of experiment 456 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cnt = cl.rad.countPamSums(456, 123) >>> cl.close()

Get PAM Sum

The "getPamSum" function returns the metadata for a PAM Sum. It takes the ids of the experiment, bucket, and pam sum as input.

Example: Get pam sum 123 of bucket 456 of experiment 789 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> ps = cl.rad.getPamSum(789, 456, 123) >>> cl.close()

Get PAM Sum Shape Grid

The "getPamSumShapegrid" function returns the data from the PAM Sum attached to the cells of the shape grid as a zipped shapefile.

Example: Get the PAM Sum shape grid for PAM Sum 123 of bucket 456 of experiment 789 and save it to /tmp/ps123.zip >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cl.rad.getPamSumShapegrid('/tmp/ps123.zip', 789, 456, 123) >>> cl.close()

Get PAM Sum Statistic

The "getPamSumStatistic" function requests a computed statistic about the PAM Sum from the server. Make sure to request that the statistics are calculated first.

Example: Get the diversity statistic 'whittakersBeta' for pam sum 123 of bucket 456 of experiment 789 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> whitBeta = cl.rad.getPamSumStatistic(789, 456, 123, 'whittakersbeta') >>> cl.close()

Get PAM Sum Statistics Keys

The "getPamSumStatisticsKeys" function returns a list of the available statistics for a PAM Sum. You can select which type of keys to return by setting the 'key' parameter to either keys, specieskeys, siteskeys, or diversitykeys

Example: Get the species statistics keys for PAM Sum 123 of bucket 456 of experiment 789 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> statKeys = cl.rad.getPamSumStatisticsKeys(789, 456, 123, keys="specieskeys")) >>> cl.close()

List PAM Sums

The "listPamSums" function returns a list of PAM Sums that match the specified parameters. Information about these parameters can be found in the code documentation.

Example: List PAM Sums randomized with the swap algorithm for bucket 123 of experiment 456 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> pss = cl.rad.listPamSums(456, 123, randomized=1, randomMethod=1) >>> cl.close()

Calculate PAM Sum Statistics

The "calculatePamSumStatistics" function tells the server that it should calculate statistics for the specified PAM Sum

Example: Request the statistics for PAM Sum 123 of bucket 456 of experiment 789 to be calculated >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cl.rad.calculatePamSumStatistics(789, 456, 123) >>> cl.close()

Compress PAM Sum

The "compressPamSum" function requests that a PAM Sum be compressed by removing empty rows and columns

Example: Compress PAM Sum 123 of bucket 456 of experiment 789 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cl.rad.compressPamSum(789, 456, 123) >>> cl.close()

Layers

Lifemapper RAD layers are spatial raster or vector layers that will be used in a macroecology experiment. They could be climate data, distribution maps, masks, or anything else.

Count Layers

The "countLayers" function counts the number of RAD layers that match the specified parameters. Information about the available parameters can be found in the code documentation.

Example: Count layers that have the name 'someName' >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> count = cl.rad.countLayers(layerName='someName') >>> cl.close()

Get Layer

The "getLayer" function allows you to retrieve metadata for a layer. This function only takes the id of the layer you would like to retrieve as input.

Example: Get layer 12345 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> lyr = cl.rad.getLayer(12345) >>> cl.close()

List Layers

The "listLayers" function allows you to return a list of layers that match the specified parameters. Information about these parameters can be found in the code documentation.

Example: List the first 4 layers with EPSG 2163 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> lyrs = cl.rad.listLayers(perPage=4, epsgCode=2163) >>> cl.close()

Post Raster Layer

The "postRaster" function allows you to post a raster layer to be used in macroecological experiments.

Example: Post a raster file named 'MyRasterLayer' located at '/layers/myRasterLayer.tif' >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> lyr = cl.rad.postRaster('MyRasterLayer', filename='/layers/myRasterLayer.tif', dataForamt='GTiff') >>> cl.close()

Post Vector Layer

The "postVector" function allows you to post a vector layer to be used in macroeclogical experiments. You can either supply a filename for a shapefile that has been zipped together or point it at the .shp file of the shapefile and it will be uploaded.

Example: Post a vector shapefile named 'MyShape' located at '/layers/myShape.shp' >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> lyr = cl.rad.postVector('MyShape', filename='/layers/myShape.shp') >>> cl.close()
Example: Post a vector shapefile named 'MyShapeZip' located at '/layers/myShape.zip' >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> lyr = cl.rad.postVector('MyShapeZip', filename='/layers/myShape.zip') >>> cl.close()

Shape Grids

Lifemapper shapegrids are discription of area of interest containing cells upon which statistics will be analyzed.

Count Shape Grids

The "countShapegrids" function counts the number of shape grids that match the criteria you specify. More information about the available parameters can be found in the code documentation.

Example: Count the number of square-shaped shapegrids for some user >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> cnt = cl.rad.countShapegrids(cellSides=4) >>> cl.close()

Get Shape Grid

The "getShapegrid" function retrieves the metadata for a shapegrid specified by its id.

Example: Get Shape Grid 12345 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> sg = cl.rad.getShapegrid(12345) >>> cl.close()

List Shape Grids

The "listShapegrids" function returns a list of shape grids that match the criteria specified. More information about the available parameters can be found in the code documentation.

Example: List the first 10 shapegrids with hexagonal cells >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> sgs = cl.rad.listShapegrids(perPage=10, cellsides=6) >>> cl.close()

Post Shape Grid

The "postShapegrid" function creates a new shape grid layer in the Lifemapper system. It takes parameters to specify the extent, cell shape and size, map projection, and even a mask if desired.

Example: Create a shapegrid with the name 'MyShapeGrid' with square cells that are 1 decimal degree in length / width in the geographic coordinate system (EPSG 4326) and global extent >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> sg = cl.rad.postShapegrid('MyShapeGrid', 'square', 1, 'dd', 4326, '-180,-90,180,90') >>> cl.close()

Other Functions

The other RAD functions are helper functions that can be useful for multiple objects.

Get Status Stage

The "getStatusStage" function returns the status and stage of objects that have those attributes

Example: Get the status and stage of bucket 123 of experiment 456 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> bkt = cl.rad.getBucket(456, 123) >>> status, stage = cl.rad.getStatusStage(bkt) >>> cl.close()
Example: Get the status and stage of pamsum 789 of bucket 123 of experiment 456 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> ps = cl.rad.getPamSum(456, 123, 789) >>> status, stage = cl.rad.getStatusStage(ps) >>> cl.close()
